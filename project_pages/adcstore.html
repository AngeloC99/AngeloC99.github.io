ADC-Store

The e-commerce that rewards you.

About The Project

ADC-Store is an e-commerce which allows the user to shop and choose the payment and shipping methodologies. The Web-app addresses the necessity of a supermarket chain to have its services digitalized, in particular in response to the pandemic emergency we lived in 2020.

Types of users:

  • Administrator: he/she handles the available goods and has access to the registered users' data.
  • Registered User: he/she can shop on the e-commerce, by gaining points to get prices, and can save his/her favorite shopping carts.
  • Not Registered User: he/she can see the product for sale and can register to access the functionalities reserved to the registered users.

The documentation produced during the project's development is available at: ADC-Store Documentation.

System Architecture

The system architecture follows the MVC pattern and it is developed in four layers:

  • View: developed in HTML+CSS+JavaScript and relying on Smarty as template engine, it is responsible for the UI, dealing with the visualization and the interaction with the user.
  • Control: it listens to events triggered by the view (or another external source) and executes the appropriate method on the model.
  • Entity (Model): it contains business logic and application data and it was developed in PHP.
  • Foundation: it deals with the persistence of the data and handle the communication with the database (MariaDB).
