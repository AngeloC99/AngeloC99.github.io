BeeWork

Help workgroup in organizing themselves productively.

About The Project

BeeWork is a mobile application created for boosting the cooperation within a team working on a project, helping the members to handle and partition the task easily and evenly. The main features are:

  • every team can create its own project's showcase, shared among the collaborators, to gather the tasks to do;
  • in every moment, a member can see the list of projects and task assigned to him;
  • tasks help the team in monitoring the state of the project, updated everytime one of them is completed.

System Architecture

The project is composed by a frontend and a backend, exchanging data through a REST API. The microservices are:

  • Frontend: it was developed using the Ionic framework, a complete and open-source SDK for the hybrid development of mobile applications, based on AngularJS and Apache Cordova.
  • Backend: the backend code was written with the help of Spring Boot, a Java framework which facilitates the creation of stand-alone applications.

The documentation produced during the project's development is available at: Frontend documentation and Backend documentation.
